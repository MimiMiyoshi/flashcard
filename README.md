単語帳アプリ

## ② 課題内容（どんな作品か）

- 登録画面で覚えたい単語を登録する
- 昔ながらのペラペラめくるタイプのシンプルな単語帳を表示させることができる
- 表面に単語を表示させることも、和文の意味を表示させることもできる
